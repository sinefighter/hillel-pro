# Getting started

1. Clone repository

   `git clone https://github.com/sinefighter/hillel-pro.git`


2. Run Docker and run:

   `docker compose up -d` or `make up`


3. Go to the php container: 

    `docker exec -it php_php-pro-4 bash` or `make php`


4. Run php-script

   `php index.php`
